Every CSS selector has its place in the specificity hierarchy.

There are four categories which define the specificity level of a selector:

1. important2. Inline styles - Example: <h1 style="color: pink;">3. IDs - Example: #navbar4. Classes, pseudo-classes, attribute selectors - Example: .test, :hover, [href]5. Elements and pseudo-elements - Example: h1, :before
