# [鐵人賽 2022-擊敗前端面試大作戰]Behavior Questions

除了技術面試題之外，很多公司在面試時還會加上 behavior questions。什麼是 behavior questions? behavior questions 是一系列的問題設計來看你對特定情境下的反應，面試者可以透過 behavior questions 來看你的回答是否符和該公司的企業文化或是徵才標準。

![5w1h](./img/5w1h.jpeg)

對象 （What）——什麼事情
場所 （Where）——什麼地點
時間和程序 （When）——什麼時候
人員 （Who）——誰
為什麼（Why）——原因
